## Research Theme
Topic 3: Probabilistic Forecasting & Uncertainty  

## Project Overview
This project proposes a probabilistic time series forecasting framework for healthcare demand prediction. Unlike traditional point forecasting approaches, this study focuses on modelling uncertainty to improve decision-making reliability in critical healthcare environments.
